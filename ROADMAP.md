# Roadmap

Planned improvements and features for this repository.

## Short Term (1–2 months)

- [ ] **Web-based Atom Browser** — Browse and search atoms in the browser
- [ ] **CI Performance Regression** — Track benchmark results across commits
- [ ] **More Demo Scripts** — NLP pipeline, query optimization, game AI
- [ ] **Jupyter Notebooks** — Interactive Python-based tutorials
- [ ] **Pre-built Docker Images** — Published to GHCR for faster setup

## Medium Term (3–6 months)

- [ ] **Python Package** — `pip install opencog-cogcli` with full CLI
- [ ] **REST API Improvements** — CRUD endpoints for atoms
- [ ] **Multi-language Examples** — More languages, better docs
- [ ] **GitHub Pages Documentation Site** — Full docs with search
- [ ] **Automated Screenshot Updates** — CI generates fresh screenshots

## Long Term (6+ months)

- [ ] **Hyperon / MeTTa Bridge** — Example of bridging Classic and Hyperon
- [ ] **Visualizer Enhancements** — Filter, search, edit in the 3D graph
- [ ] **Knowledge Graph Importers** — RDF, JSON-LD, CSV → AtomSpace
- [ ] **Performance Benchmarks** — Full suite comparing storage backends
- [ ] **Community Templates** — Issue/PR templates for OpenCog upstream
