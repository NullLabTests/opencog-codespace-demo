# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-06-01

### Added

- 17 comprehensive documentation files covering getting-started, architecture,
  Atomese primer, knowledge patterns, integrations, API, vocabulary, glossary,
  best practices, FAQ, ecosystem, history, neuro-symbolic AI, performance guide,
  migration guide, build-from-source, and visualizer pages
- 23 demo scripts including comprehensive-demo, taxonomy, family-tree,
  expert-system, commonsense, planning, temporal, semantic-net, analogies,
  math-reasoning, rewriting, chatbot-memory, recipe-knowledge, music-theory,
  geography, diagnostics, evaluation-demo, stress-test, truth-values,
  values-api, rest-api, game-rule-engine, sensor-integration, and
  query-optimization
- Python CLI tool (`cogcli.py`) with 7 subcommands
- Async load tester (`loadtest.py`)
- Dockerfile and docker-compose.yml for containerized deployment
- Makefile with start, stop, build, and test targets
- GitHub Actions CI pipeline (build + test on every push)
- Community files: CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, CONTRIBUTORS
- Apache 2.0 license
- Dependabot configuration for automated dependency updates
- Pre-commit hook configuration
- 8 screenshots + 1 animated REPL demo GIF
- SVG architecture diagram
- GitHub Pages site with quick-start landing page
